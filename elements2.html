  • GitHub

    • GitHub is a great tool for those starting out in programming. It allows you to host projects online for free, although there are some conditions to be aware of. For example, there are limitations on space or request numbers, but it’s still more than enough to get started. Moreover, it makes collaboration with other developers easy, tracks project changes, and provides a history of all modifications. All in all, it’s a must-have for anyone wanting to share and manage their code online!

    Boxicons

    Boxicons is a fantastic site for developers looking for icons to include in their projects. It offers a large collection of SVG icons, perfect for adding lightweight, scalable graphical elements to your sites or apps. The site is simple to use: you can easily search for the icon you need, customize it, and integrate it directly into your code. It's a practical and efficient tool to enrich the user interface without weighing down your project’s performance. In short, Boxicons is an essential for anyone looking for modern, easy-to-use, and free icons!

    Symbl

    As part of my web development learning, I discovered an extremely useful tool: symbl.cc. This site helps me easily find Unicode symbols and integrate them into my HTML code. Whether it’s for adding icons, special characters, or graphic symbols to my projects, symbl.cc greatly simplifies the process.

    The site is easy to use: just type a keyword like "Speed", and it suggests a list of related symbols with their HTML or Unicode codes. This saves me from having to manually search for each code or memorize complex references. In just a few clicks, I can copy a symbol and add it directly into my code, making the process both quick and convenient.

    MySQL

    As part of my learning and integration of a database, I used MySQL, and I find it to be an excellent tool for connecting and managing databases in a simple and efficient way. Thanks to its popularity and robustness, MySQL offers a reliable and high-performance solution for creating, manipulating, and querying databases.

    One of MySQL's major advantages is its ability to integrate easily into applications, whether using languages like PHP, Python, or JavaScript. It handles SQL queries efficiently and provides excellent performance even with large datasets. In short, MySQL is an essential choice for anyone developing an application with a reliable and accessible database.
