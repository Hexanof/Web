Bienvenue sur mon portfolio ! Ce site présente mes projets de développement web et met en avant mes compétences dans plusieurs domaines du développement. Vous y trouverez des exemples de mon travail, des technologies que j'utilise, ainsi que des informations sur mes réalisations.

## Compétences

### HTML (HyperText Markup Language)
HTML est la base de tout site web. C'est le langage de balisage utilisé pour structurer le contenu sur le web. Dans mes projets, j'utilise HTML5 pour garantir une structure propre, sémantique et accessible. Que ce soit pour des pages simples ou des applications web complexes, HTML est au cœur de chaque projet que je développe.

### CSS (Cascading Style Sheets)
CSS est essentiel pour la mise en forme de pages web. J'utilise CSS pour rendre mes sites esthétiques et responsives, en m'assurant qu'ils s'adaptent parfaitement à différents écrans (ordinateurs, tablettes, téléphones). En plus des techniques classiques comme Flexbox et Grid, je maîtrise également les préprocesseurs CSS comme SASS pour un développement plus rapide et maintenable.

### JavaScript
JavaScript est le langage qui permet d'ajouter des interactions dynamiques à une page web. Grâce à JavaScript, je crée des fonctionnalités comme des formulaires interactifs, des animations, des API interactives et des applications web plus riches. J'utilise également des frameworks et bibliothèques populaires comme React et jQuery pour rendre le développement plus efficace et modulaire.

### PHP (Hypertext Preprocessor)
PHP est un langage de programmation côté serveur que j’utilise pour créer des sites web dynamiques et interagir avec des bases de données. Je l’utilise pour construire des applications web, des systèmes de gestion de contenu (CMS), et des solutions backend robustes. J'ai de l'expérience avec des outils comme MySQL pour la gestion des bases de données et j'utilise PHP pour créer des sites évolutifs et sécurisés.
